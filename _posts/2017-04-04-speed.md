# 1. 自加和赋值加法的效率比较

```python
import time

n=100000
start=time.time()
i=1
for j in range(n):
    i+=1
print(time.time()-start)

start=time.time()
i=1
for j in range(n):
    i=i+1

print(time.time()-start)
```


>0.01636791229248047
0.01322317123413086
